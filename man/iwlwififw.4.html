Firmware modules for Intel iwlwifi wireless network driver

SYNOPSIS

The iwlwifi(4) driver should auto-load any firmware needed. It is discouraged to load the driver or firmware manually from loader(8).

DESCRIPTION

Multiple modules provide access to firmware for the various chipset models supported by the iwlwifi(4) driver.
