Setting up a development environment with aqua-unix

aqua-unix is a script which automatically downloads, compiles, and installs all the necessary components for developing AQUA applications and devices on UNIX-based systems. You do not necessarily have to be running an aquaBSD system for this to work - many other closely POSIX-compliant systems such as Linux will do fine too - but it is recommended for later steps.

For full documentation, please consult the README.md file on the official GitHub repository.

To download aqua-unix and compile the KOS, aquabsd-manager, iar, and the devices from the aquabsd.alps devbranch, run:

% git clone https://github.com/inobulles/aqua-unix
% cd aqua-unix
% sh build.sh --kos --manager --devices --devbranch aquabsd.alps --install
