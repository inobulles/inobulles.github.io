bsddialog(1)

BSDDIALOG(1)FreeBSD General Commands ManualBSDDIALOG(1)

NAME

bsddialog —
TUI dialogs

SYNOPSIS

bsddialog--help

bsddialog--version

bsddialog[--common-option] --dialogtextrowscols [dialog-parameter]

DESCRIPTION

The bsddialog utility builds Text User Interface dialogs and widgets: to display messages, to get input and to inform about a computation status.

The options --help and --version print the list of options and the version, respectively, then exit.

The following options are available for each dialog. text is a message printed inside the dialog, except for --textbox described later. rows and cols are the height and width, 0 for autosize and -1 for fullscreen. The possible input got from the user interface is printed to standard error.

Common options

The following options can change the default behavior of the utility and are common to some dialog.
--ascii-lines
Ascii characters to draw lines.
--backtitlebacktitle
Title on the top side of the screen.
--begin-xx
Dialog horizontal position, 0 is the left screen side, -1 center.
--begin-yy
Dialog vertical position, 0 is the top screen side, -1 center.
--cancel-labellabel
Label for the “Cancel” button.
--clear
Hide the dialog at exit.
--colors
Enable highlights for text, the following sequences are considered escapes:
“\Z0” black.
“\Z1” red.
“\Z2” green.
“\Z3” yellow.
“\Z4” blue.
“\Z5” magenta.
“\Z6” cyan.
“\Z7” white.
“\Zr” reverse foreground and background colors.
“\ZR” disable reverse.
“\Zb” bold.
“\ZB” disable bold.
“\Zu” underline.
“\ZU” disable underline.
“\Zn” restore to normal text.
--cr-wrap
Replace new line with a space in text.
--date-formatformat
String accepted by strftime(3) to customize the output of --datebox.
--default-buttonlabel
Focus on the button with label on startup.
--default-itemname
Focus on the item with name, for Checklist, Menu, Radiolist and Treeview.
--default-no
Focus on “Cancel” or “No” button on startup.
--defaultno
Equivalent to --default-no.
--disable-esc
Disable ESC key to quit.
--esc-cancelvalue
Exits with the “Cancel” button value if the ESC key is pressed.
--exit-labellabel
Equivalent to --ok-label.
--extra-button
Add a button with “Extra” label.
--extra-labellabel
Set label for the “Extra” button.
--generic-button1label
Add a button with label.
--generic-button2label
Add a button with label.
--help-button
Add a button with “Help” label.
--help-labellabel
Set label for “Help” button.
--help-status
Print also the selected items if the “Help” button is pressed.
--help-tags
Print the name of the focused item if the Help button is pressed also with --item-help.
--hfilefilename
Open filename in a Textbox if F1 key is pressed,
--hlinestring
Dialog subtitle.
--hmsgstring
Open a Msgbox with string if the F1 key is pressed.
--ignore
Do not exit with unknown options.
--insecure
Print ‘*’ to hide passwords while typing, default space ‘ ’.
--item-depth
Specify a margin for items, available for Checklist, Menu and Radiolist.
--item-help
Set a help string for each element of a Checklist, Form, Menu, Mixedform, Passwordform, Radiolist and Treeview to display at the bottom screen side.
--item-prefix
Set a string to prefix each item of a Checklist, Menu, Radiolist or Treeview.
--max-inputsize
Maximum length of the input for --input-box ans --passwordbox, default 2048.
--no-cancel
Do not show “Cancel” button.
--no-collapse
Do not replace a TAB character with a space in text.
--no-items
Do not display items desciption, for Checklist, Menu, Radiolist or Treeview.
--no-labellabel
Equivalent to --cancel-label.
--no-lines
Do not draw borders and lines.
--no-nl-expand
do not consider the sequence “\n” like new line.
--no-ok
Do not draw “OK” button.
--no-shadow
No not draw the shadow of the dialog.
--no-tags
Do not display items name, for Checklist, Menu and Radiolist.
--nocancel
Equivalent to --no-cancel.
--nook
Equivalent to --no-ok.
--ok-labellabel
Set label for “OK” button.
--output-fdfd
Print input from user interface to the specified file descriptor.
--output-separatorsep
Set a sepator for the items in output, default whitespace.
--print-maxsize
Screen size.
--print-size
Print Dialog height and widget at exit.
--print-version
Print version.
--quoted
Quote items in output, default only when necessary.
--separate-output
Separate selected items with a new line and avoid to quote.
--separatorsep
Equivalent to --output-separator.
--shadow
Show a pseudo shadow for the dialog, enabled by default.
--single-quoted
Use single quote for items in output.
--sleepsecs
Wait secs seconds to close the dialog.
--stderr
Print input from user interface to standand error, default.
--stdout
Print input from user interface to standard output.
--tab-lenspaces
Number of spaces to print a TAB in text.
--themetheme
Set a graphical style: blackwhite, bsddialog, flat or dialog.
--time-formatformat
String accepted by strftime(3) to customize the output of --timebox.
--titletitle
Dialog title.
--trim
remove consecutive spaces in text.
--yes-labellabel
Equivalent to --ok-label.

Dialogs

The following dialogs are available:
--checklisttextrowscolsmenurows [namedescstatus] ...
Checklist to select some item from a list via the SPACE key. An item has a name, desc and a default status specified by “on” or “off”. The names of the selected items are printed to standard error. menurows is the graphical height of the list, 0 for autosize.
--dateboxtextrowscols [yearmonthday]
Dialog to select a date.
--formtextrowscolsformrows [labelylabelxlabelinityfieldxfieldfieldlenmaxvaluelen] ...
Dialog to get a list of strings via forms. A form has a label at the position ylabel and xlabel, a field to get the input at the position yfield and xfield with graphical length fieldlen, maxvaluelen is the maximum input length. The field can be customized, if fieldlen is 0 its length is the absolute value of maxvaluelen, if maxvaluelen is negative the field is read only, init is a default value. formrows is the graphical height of the list, has to be at least the number of forms.
--gaugetextrowscols [percentage]
Dialog with a bar to shows percentage, then it waits to read “XXX” from the standard input, then the first string replaces percentage and the following strings replace text until the next “XXX”, the loop ends reading EOF.
--infoboxtextrowscols
Dialog without buttons to display a message and to exit immediately.
--inputboxtextrowscols [init]
Dialog to get a string in input, init is the default value.
--menutextrowscolsmenurows [name desc] ...
Builds a menu to select an item from a list. An item has a name and a desc. The name of the selected item is printed to standard error. menurows is the graphical height of the list, 0 for autosize.
--mixedformtextrowscolsformrows [labelylabelxlabelinityfieldxfieldfieldlenmaxvaluelenflag] ...
Dialog to get a list of strings via forms. A form has a label at the position ylabel and xlabel, a field to get the input with graphical length fieldlen at the position yfield and xfield, maxvaluelen is the maximum input length, init is a default value, flag can be 0 for normal field, 1 to hide the typed characters and 2 to set the field read only. formrows is the graphical height of the list, has to be at least the number of forms.
--mixedgaugetextrowscolsmainperc [minilabelminiperc] ...
Dialog to show a main bar to represent mainperc from 0 to 100 and some mini bar with a minilabel string and a miniperc with value from 0 and 100 or negative to print a descriptive string: -1 “Succeeded”, -2 “Failed”, -3 “Passed”, -4 “Completed”, -5 “Checked”, -6 “Done”, -7 “Skipped”, -8 “In Progress”, -9 a blank line, -10 “N/A”, -11 “Pending”.
--msgboxtextrowscols
Dialog to diplay a message without the “Cancel” button.
--passwordboxtextrowscols [init]
Dialog to get a password, init is the default value.
--passwordformtextrowscolsformrows [labelylabelxlabelinityfieldxfieldfieldlenvaluelen] ...
Dialog to get a list of passwords, equivalent to --form except typed characters are hidden.
--pausetextrowscolsseconds
Dialog runs until the timeout in seconds expires or a button is pressed.
--radiolisttextrowscolsmenurows [namedescstatus] ...
Radiolist to select at most an item from a list via the SPACE key. An item has a name, desc and a default status specified by “on” or “off”. The name of the selected item is printed to standard error. menurows is the graphical height of the list, 0 for autosize.
--rangeboxtextrowscolsminmax [init]
Dialog to select a value between min and max, init is the default value, the keys UP, DOWN, HOME, END, PAGEUP and PAGEDOWN can change it.
--textboxfilerowscols
Opens and prints file the UP, DOWN, HOME, END, PAGEUP and PAGEDOWN keys are availble to navigate; “OK” button is renamed “EXIT”.
--timeboxtextrowscols [hourminsec]
Dialog to select a time.
--treeviewtextrowscolsmenurows [depthnamedescstatus] ...
Equivalent to Radiolist with --item-depth and --no-name.
--yesnotextrowscols
“Yes-No Question”, “OK” and “Cancel” buttons are renamed “Yes” and “No”.

EXIT STATUS

The bsddialog utility exits 255 on unsuccessful, otherwise depending on the button or key pressed the following values can be returned:
0 “OK”, “Yes” or “Exit” button.
1 “Cancel” or “No” button.
2 “Help” button.
3 “Extra” button.
4 Timeout.
5 ESC key.
6 Generic 1 button.
7 Generic 2 button.

EXAMPLES

Backtitle, title and message:
bsddialog --backtitle MESSAGE --title Msgbox --msgbox Message 0 0

Yes-No Question and theme:

bsddialog --theme blackwhite --yesno Question 10 30

Checklist:

bsddialog --checklist Checklist 0 0 3 N1 D1 off N2 D2 on N3 D3 off

Mixedgauge:

bsddialog --sleep 3 --mixedgauge Example 10 30 60 L1 " -1" L2 30

Form:

bsddialog --form Form 0 0 2 L1: 1 1 X 1 5 20 25 L2: 2 1 X 2 5 20 25

Gauge script:

i=1
for c in A B C D E F G H
do
	sleep 1
	echo XXX
	echo "$(expr $(expr $i "*" 100) "/" 8)"
	echo "[$i/8] Char: $c"
	echo XXX
	if [ $i -eq 8 ]
	then
		sleep 1
		echo EOF
	fi
	i=`expr $i + 1`
done | bsddialog --title Gauge --gauge "Starting..." 10 70

Mixedgauge script:

perc=0
while [ $perc -le 100 ]
do
	bsddialog --sleep 1 --title Mixedgauge          \
		--mixedgauge "\nExample...\n" 0 0 $perc \
		"Hidden"   " -9"                        \
		"Label 1"  " -4"                        \
		"Label 2"  " -4"                        \
		"Label 3"  $perc

	perc=`expr $perc + 20`
done
